Example: Zeolites and Their Properties

Zeolites are fascinating materials used in many fields like catalysis and gas separation. Their performance often depends on their structure, which in turn is influenced by the arrangement of atoms. In this example, we explore how Germanium (Ge) atoms distribute within a zeolite structure and how that affects their stability.

Interactive Code: Build a Simple Regression Model

Below is a regression model where you can input data and see how well it predicts outcomes. We’re predicting the energy of a zeolite structure based on its Ge-O-Ge atom counts.


# Simple regression model example in Python (sklearn)
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression

# Sample data: Ge-O-Ge counts and corresponding energies
X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
y = np.array([3, 4, 2, 5, 7])

# Initialize the model
model = LinearRegression()
model.fit(X, y)

# Predict
new_X = np.array([input("Enter a new Ge-O-Ge count: ")]).reshape(-1, 1)
predicted_energy = model.predict(new_X)

# Plot the regression line
plt.scatter(X, y, color='blue')
plt.plot(X, model.predict(X), color='red')
plt.xlabel('Ge-O-Ge Count')
plt.ylabel('Energy')
plt.title('Energy vs Ge-O-Ge Count')
plt.show()

print(f"The predicted energy for Ge-O-Ge count {new_X[0][0]} is {predicted_energy[0]}")

Try adjusting the inputs and observe how the energy changes. You'll also see the linear regression fit visually!

Real Data Visualization: UMAP Plots of Energy vs Ge-O-Ge Count

Now that you’ve seen a simple example, let's explore real scientific data. We use UMAP (Uniform Manifold Approximation and Projection) to visualize relationships between energy and Ge-O-Ge counts in real zeolite structures.

UMAP helps us group similar structures and observe patterns that might otherwise be difficult to find. This data visualization shows how the distribution of Ge atoms affects the stability of zeolites.

Machine Learning Insights: Energy Prediction

Using real data, we see how machine learning provides insights that help predict the stability of new zeolite structures before they are even synthesized in a lab. By training models on existing data, we can forecast how changing atomic arrangements will affect energy and stability.

The regression model you interacted with earlier is just one example. In research, more advanced models like neural networks and decision trees are used to predict material properties with higher accuracy.
